# Cinemaplay
It's basic home page of a online streaming site.
A landing page containing Login and Signup using a popup logic, a Image carousel,and different rows displaying movie cards of different medias like Trending,Movies,Tv shows and anime.
A container below Trending row which has a genres based sorting logic using genres buttons of different category . on click event of this buttons will sort those category movies and display them.
At last their is a simple footer.
